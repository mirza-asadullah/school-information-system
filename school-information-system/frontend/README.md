# Frontend

This frontend app is a React + Vite scaffold using Material UI, Redux Toolkit, React Router, and Axios.

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

## Structure

- `src/api/` - API client and HTTP helpers
- `src/components/` - reusable UI building blocks
- `src/pages/` - route-specific screen components
- `src/layouts/` - UI layout wrappers
- `src/routes/` - central route definitions
- `src/store/` - application state configuration
- `src/hooks/` - reusable React hooks
- `src/utils/` - shared utilities
- `src/theme/` - Material UI theme configuration
